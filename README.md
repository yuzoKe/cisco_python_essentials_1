## 🎮 Final Project – Tic-Tac-Toe

A command-line Tic-Tac-Toe game where the computer plays as `X`
and the user plays as `O`. The computer always starts in the center.

**Concepts applied:**
- Modular functions
- Nested lists (2D board)
- Input validation with `try/except`
- Win detection across rows, columns and diagonals
- Random move generation with `randrange`
